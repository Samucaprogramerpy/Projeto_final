# **Aplicativo de Gerenciamento de Limpezas** # 

#### Este é o aplicativo de Gerenciamento de Limpezas das salas do Senac RN desenvolvido, desenvolvido com React Native usando Typescript. Ele é acompanhado de um sistema de autenticação com token e fornece informações, como o nome das salas, status de limpeza, status de atividade, capacidade, localização, ultimo horário de limpeza, ultimo funcionário a limpar a sala, informções de usuários (para os administradores) e entre outras informações. ###
